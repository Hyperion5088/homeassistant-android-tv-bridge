# Changelog

## 0.1.0-beta.1

Initial public beta.

- Combined Android TV Remote and ADB control in one Home Assistant media player.
- Added app launching, local source switching, power control, volume control, and media metadata sensors.
- Added configurable polling and source/app filter settings.
- Added friendly source naming for local inputs where device data is available.
- Added ADB startup warm-up, transient reconnect handling, and post-power-on Home wake nudge.
- Added HACS metadata, Hassfest/HACS validation workflows, brand assets, MIT license, and dependency notices.
